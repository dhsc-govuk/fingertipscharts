# Fingertips theme for ggplot2

fingertips theme

## Usage

``` r
theme_phe(
  theme = "fingertips",
  base_size = 11,
  base_family = "",
  base_line_size = base_size/22,
  base_rect_size = base_size/22
)
```

## Arguments

- theme:

  string; theme of chart, current are fingertips only

- base_size:

  base font size

- base_family:

  base font family

- base_line_size:

  base size for line elements

- base_rect_size:

  base size for rect elements
