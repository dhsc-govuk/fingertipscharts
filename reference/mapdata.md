# Dummy data for Upper Tier Local Authorities so dummy maps can be produced

A dataset of indicator values for all Upper Tier Local Authorities

## Usage

``` r
mapdata
```

## Format

A data frame with 152 records and 4 fields:

- AreaCode:

  Office for National Statistics area codes

- AreaName:

  Office for National Statistics area names

- Significance:

  The statistical significance of the area compared to a benchmark

- Value:

  The indicator value for the data
